Скачущий Мячик
Програма симульятор как скачет мячик на экране. Написана эта програма с помошю
языка (Хорошый Учебный Язык) или можно кратко - ХУЯ. Такой уникальный язык был
разработан Алексейом Кутёповым каторый обычно известный под именами Tsoding или Rexim.

Сыллка на компайлер (https://github.com/tsoding/good_training_language.git)

Как пользаваться?
в файле запуск.ш надо поменять бкржуйскую переменную COMPILER_DIR чтобы эта 
переменная правильно указывала в папку где находится компайлер ХУЯ, убедитесь
что сам эксэ файл находится в указанной папке! Скрипт запуск.ш сам скомпилирует
и запустит програму. Скомпилированный файл будет под названнием ./главный
